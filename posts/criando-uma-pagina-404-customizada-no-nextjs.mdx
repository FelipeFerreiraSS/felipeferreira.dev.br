Todo site na internet conta com uma página 404 e se você desenvolveu o seu utilizando Next.js, provavelmente tem uma página genérica, já programado pelo framework. Nesse artigo ensinarei como é simples criar uma página de erro 404 personalizada para seu site.

Criar uma página 404 no Next.Js é super simples, basta adicionar um arquivo com o nome <QuoteCode code={`404.js`}/> na pasta <QuoteCode code={`pages`}/> do seu projeto. 

Este é um exemplo de como pode ser feita a sua página 404:

<Code
code={
`import Link from  'next/link'  
  
export  default  function  Error404()  {  
	return(  
		<main>
			<h1>404</h1>
			<div>
				<h2>Ooops!</h2>
				<h2>Página não encontrada</h2>
			</div>
				
			<p>Esta página não existe ou foi removida!</p>
			<p>Recomendo que volte para a Home</p>
				
			<Link  href={"/"}>
				<button>Home</button>
			</Link>
		</main> 
	)  
}
`}  language="js"  />

Está pagina é como qualquer outra no Next.js. Então você pode estilizar como quiser, colocando imagens, animações e o que mais fazer sentido para seu site.
