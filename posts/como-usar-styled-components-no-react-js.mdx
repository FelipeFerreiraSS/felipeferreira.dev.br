## O que é Styled Components ?

Styled Components é uma biblioteca (lib) que permite escrever código CSS dentro do Java Script também conhecido como CSS-in-JS, ganhando mais agilidade na criação de aplicações web e mobile.

## Configurando projeto React JS

Primeiro criaremos um projeto com React JS usando o <QuoteCode code={`create-react-app`}/>. Execute esse comando no seu terminal, dentro da pasta de sua escolha.

<Code
code={
`   npx create-react-app my-app
`}  language="json"  />

Após terminar o processo, teremos um projeto com uma estrutura de pastas parecida com essa:

Vamos apagar alguns arquivos que não precisamos mais como: <QuoteCode code={`App.css`}/>, <QuoteCode code={`App.teste.js`}/>, <QuoteCode code={`index.css`}/>, <QuoteCode code={`logo.svg`}/>, <QuoteCode code={`serviceWorker.js`}/> e <QuoteCode code={`setupTests.js`}/> no final o seu projeto terá uma estrutura parecida com essa:

<ImagePost  id="/assets/arquivos-removidos.png"  alt="teste de imagem"/>

Apos apagar os arquivos começaremos o projeto, no seu arquivo <QuoteCode code={`App.js`}/> apague algumas linhas de código até ficar assim:

## Utilizando Styled Components exemplo

Para usar o styled components vamos primeiro criar um arquivo <QuoteCode code={`styles.js`}/> na pasta <QuoteCode code={`src`}/>. Após criar o arquivo adicione o seguinte código: 

```js
import  styled  from  'styled-components'

export  const  Title = styled.h1`
	color: #0000FF;
`
```

Esse código importa o styled-components e cria um componente com o nome <QuoteCode code={`Title`}/>, que sera usado para estilizar um título, como podemos ver na tag HTLM escrita depois de styled: <QuoteCode code={`styled.h1`}/>. Depois  adicione o código CSS,  que nesse casso só alteraremos a cor do título. 

Vá para o <QuoteCode code={`App.js`}/> e importe o <QuoteCode code={`Title`}/> do arquivo <QuoteCode code={`styles`}/>, depois adicione como se fosse uma tag HTML

```js
import { Title } from  './styles'
 
function  App() {
	return (
		<div>
			<Title>Hello world</Title>
		</div>
	);
}

export  default  App;
```
Salve o arquivo e volte para o navegador e você perceberá que o texto mudou de cor para azul.
