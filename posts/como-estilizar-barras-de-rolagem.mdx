A barra de rolagem é sempre vista em praticamentos todos os sites da internet e adicionar um estilo a ela pode fazer muita diferença no design do seu site.

Neste artigo eu te mostro que estilizar a barra de rolagem é super simples e deixará seu site com muito mais elegante. 

## Como funciona a barra de rolagem
Antes de estilizar a barra de rolagem é importante entender como ela funciona para depois partimos para o CSS.
para estilizar a barra utilizamos um pseudo-elemento <QuoteCode code={`-webkit-scrollbar`}/>

-   <QuoteCode code={`::-webkit-scrollbar`}/>  é a barra principal.
-   <QuoteCode code={`::-webkit-scrollbar-button`}/>  são os botões na barra de rolagem, as setinhas que apontam para cima e para baixo.
-   <QuoteCode code={`::-webkit-scrollbar-thumb`}/>  é a barra de rolagem arrastável.
-   <QuoteCode code={`::-webkit-scrollbar-track`}/>  é a faixa (barra de progresso) da barra de rolagem.
-   <QuoteCode code={`::-webkit-scrollbar-track-piece`}/>  é a faixa (barra de progresso) que não é coberta pela alça.
-   <QuoteCode code={`::-webkit-scrollbar-corner`}/>  é o canto inferior da barra de rolagem, onde as barras de rolagem horizontal e vertical se encontram.
-   <QuoteCode code={`::-webkit-resizer`}/>  é o identificador de redimensionamento arrastável que aparece no canto inferior de alguns elementos.

## Estilizando barra de rolagem
